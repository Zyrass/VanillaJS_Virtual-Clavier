Reproduction d'un clavier en Vanilla JS

Description

J'ai reproduit un clavier pour pratiquer le javaScript (VanillaJS). Je sais que je ne respecte pas les bonnes pratiques dans le sens où je me répète à 90% du temps. J'insiste sur un point important : Je suis débutant. Mais pour le moment le résultat est lui plutôt sympatique et surtout, fonctionnel. Attention, je ne suis en aucun cas contre le faite de l'améliorer.

Volontairement, les touches Shift(s) ne permettent pas de créer les majuscules.
En effet je préfère laisser cette fonctionnalitée avec la touche "Verr Maj A" aussi connu sous le nom de Caps Lock.
